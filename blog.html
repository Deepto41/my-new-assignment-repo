Question-1: What are the different ways to select an element in the DOM?
We can select an element in the dom in different ways like:
1.Using ID: document.getElementById();
2.Using Class Name : document.getElementsByClassName();
3.Using Tag Name : document.getElementsByTagName();
4.Using Query Selector : document.querySelector();
5.Using Query Selector All : document.querySelectorAll();
Question-2: What is the difference between innerHTML, innerText, and textContent ?
1.By using innerHTML,we can find the HTML content of an element.
2.By using innerText, we can find the visible text of an element.
3.By using innerContent,we can find the text content of an element.
Question-3: What is event delegation in the DOM?
Event delegation is a technique in JavaScript where we use a single event listener to a parent element instead of multiple.
Question-4: What is event bubbling in the DOM?
Event bubbling is a behavior in the DOM where an event starts from the target element and end to the parent element in the DOM tree.
Question-5: How do you create, add, and remove elements using JavaScript?
We can create a new element using - document.createElement().
We can add an element by using appendChild() .
we can remove an element by using .remove() .
